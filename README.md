
# Class-DigitalImageProcessing

원광대학교 게임콘텐츠학과 **게임과영상처리** 강의용 GitHub 저장소입니다.  
본 저장소는 디지털 영상처리의 핵심 이론과 이를 실제 코드로 구현하는 실습 자료를 함께 다루기 위해 구성되었습니다.

강의에서는 디지털 영상의 표현과 저장, 화질 개선, 주파수 변환, 필터링, 특징 검출, 형태학 기반 처리, 영상 압축 등 영상처리의 주요 주제를 학습하며, C / C# / OpenCVSharp 기반 실습을 통해 알고리즘을 직접 구현합니다. 

---

## 🛠️ 강의 개요

- **교과목명**: 게임과영상처리
- **대상**: 원광대학교 게임콘텐츠학과 2학년
- **학점**: 3학점
- **수업 형태**: 이론 + 실습 연계형 강의
- **강의 운영**: 오프라인 중심
- **주요 실습 환경**: C, C#, OpenCVSharp
- **강의자료 배포**: Waffle
- **교재**: *이론과 실습으로 배우는 디지털 영상처리* (한빛미디어)
- **참고도서**: *Digital Image Processing* (곤잘레스) 

본 강의는 디지털 영상처리의 기초 개념부터 실제 응용까지를 다루며, 학생들이 영상처리 알고리즘을 이해하는 데 그치지 않고 **직접 프로그래밍으로 구현할 수 있는 수준**에 도달하는 것을 목표로 합니다. 

---

## 🌟 강의 목표

본 강의의 목표는 다음과 같습니다.

1. 컴퓨터를 통해 처리되는 **디지털 영상의 특성**을 이해한다.
2. 영상처리를 위한 다양한 **기초 알고리즘**을 학습한다.
3. 디지털 영상의 **활용 분야와 응용 방법**을 이해한다.
4. 영상처리 알고리즘을 **프로그래밍으로 구현**할 수 있다.
